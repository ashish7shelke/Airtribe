Project description
    LearnTrack is a console-based Student & Course Management System built using Core Java.
    It will allow admins to manage:
        Students
        Courses
        Enrollments
    
    **Development is in Linux environment so documents are in basic text fromat.

How to compile and run:
    Open a terminal and run below command.
    java src/com/airtribe/learntrack/ui/Main.java
